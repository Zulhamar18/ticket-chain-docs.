Proyek Ticket-Chain - Dokumentasi SDK

Referensi Client SDK

Inisialisasi

Tambahkan tag script berikut ke semua halaman tempat Anda perlu memanggil Pi Apps SDK:

Objek konfigurasi yang dilewatkan ke fungsi init menerima kunci berikut:

  • version (string, wajib): Memastikan kompatibilitas aplikasi dengan versi SDK yang lebih baru.
  • sandbox (boolean, opsional): Mengonfigurasi SDK untuk berjalan dalam mode sandbox.

Autentikasi

Gunakan metode berikut untuk mengautentikasi pengguna:

Pi.authenticate(scopes: Array, onIncompletePaymentFound: Function): Promise

Hasil yang dikembalikan:

type AuthResult = {
  accessToken: string;
  user: {
    uid: string;
    username: string;
  };
};

Pembayaran

Buat pembayaran baru menggunakan metode berikut:

Pi.createPayment(paymentData: PaymentData, callbacks: PaymentCallbacks): void;

Iklan (Ads)

SDK menyediakan modul Ads untuk menampilkan iklan kepada pengguna. Contoh:

Pi.Ads.showAd(adType: "interstitial" | "rewarded"): Promise;
Catatan: Pastikan untuk memeriksa ketersediaan iklan menggunakan Pi.Ads.isAdReady() sebelum menampilkannya.

Fitur Lainnya

Anda juga dapat membuka dialog berbagi menggunakan:

Pi.openShareDialog(title: string, message: string): void;
